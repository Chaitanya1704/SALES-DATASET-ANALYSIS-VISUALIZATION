Sales Data Analytics Dashboard

 Project Overview
This project is a complete Sales Data Analytics Dashboard developed using SQL, Excel, Python, and Power BI. The objective is to analyze sales performance, revenue trends, profit margins, and regional growth to generate meaningful business insights.



 Tools & Technologies Used
- SQL – Data extraction and querying
- Excel – Data cleaning and preprocessing
- Python – Data analysis
- Power BI – Data visualization and dashboard creation



 Key Business Insights
- Total Revenue and Profit Analysis
- Monthly Sales Trends
- Regional Sales Performance
- Top Performing Products
- Category-wise Sales Breakdown
